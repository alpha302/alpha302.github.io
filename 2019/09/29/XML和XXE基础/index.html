XML简介

  • XML 被设计用来传输和存储数据。
  • HTML 被设计用来显示数据。

XML语法

所有XML都必须要有关闭标签。

所有XML的标签对大小写敏感

XML 标签对大小写敏感。在 XML 中，标签 与标签 是不同的。
必须使用相同的大小写来编写打开标签和关闭标签

XML的属性值必须加引号

在 XML 中， 5个实体引用来代替符号：

实体引用 符号 中文解释
&it ＜ 小于号
&gt ＞ 大于号
&amp & 和号
&apos ‘ 单引号
&quot “ 双引号
